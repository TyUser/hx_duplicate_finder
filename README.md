# hx_duplicate_finder

Эта утилита на Rust предназначена для поиска и удаления дубликатов файлов в указанной директории. Она использует хэширование (SHA-256) для сравнения файлов и может автоматически удалять дубликаты в зависимости от настроек.

## Возможности
- **Поиск дубликатов:** Сканирует директории и находит файлы с одинаковым содержимым.
- **Хэширование:** Использует SHA-256 для точного сравнения файлов.
- **Исключения:** Поддерживает списки исключений для директорий, файлов и расширений.
- **Удаление дубликатов:** Может автоматически удалять дубликаты, перемещая их в корзину.
- **Логирование:** Ведёт лог всех операций в файл `process.log`.

## Сборка из исходников
- Установите [Rust](https://www.rust-lang.org/).
- Клонируйте репозиторий и перейдите в директорию проекта:
- Соберите проект:
 ```bash
   cargo build --release
   ```
- Исполняемый файл `hx_duplicate_finder.exe` появится в папке `hx_duplicate_finder\target\release`.

## Использование
1. Поместите `hx_duplicate_finder.exe` в директорию, которую необходимо просканировать, и запустите файл.
2. Программа проанализирует текущую папку и все вложенные директории.
3. При первом запуске программа создаст папку с настройками: `C:\Users\<Имя_пользователя>\Documents\hx_settings`:
- **folders.txt** — список исключаемых директорий.
- **files.txt** — список исключаемых файлов.
- **extensions.txt** — список разрешенных расширений.
- **delete.txt** — режим работы. По умолчанию `null` (Режим: только просмотр). Для активации удаления замените на `yes`.

## Важные примечания
- **Платформа:** Windows 10/11.
- **Безопасность:** По умолчанию программа работает в режиме "только просмотр". Удаление включается только через `delete.txt`.
- **Восстановление:** Все удаленные файлы перемещаются в корзину, что позволяет восстановить их при ошибке.
- **Производительность:** Скорость сканирования зависит от скорости работы диска и объема данных.

# hx_duplicate_finder

Эта утилита на Rust предназначена для поиска и удаления дубликатов файлов в указанной директории. Она использует хэширование (SHA-256) для сравнения файлов и может автоматически удалять дубликаты в зависимости от настроек.

## Возможности

- **Поиск дубликатов:** Сканирует директории и находит файлы с одинаковым содержимым.
- **Хэширование:** Использует SHA-256 для точного сравнения файлов.
- **Исключения:** Поддерживает списки исключений для директорий, файлов и расширений.
- **Удаление дубликатов:** Может автоматически удалять дубликаты, перемещая их в корзину.
- **Логирование:** Ведет лог всех операций в файл `process.log`.

## Сборка из исходников

- Убедитесь, что у вас установлен [Rust].
- Клонируйте репозиторий.
- Соберите проект: **cargo build --release**
- Собранный **hx_duplicate_finder.exe** будет лежать в: **hx_duplicate_finder\target\release**

## Использование

1. Поместите программу **hx_duplicate_finder.exe** в место где имеется множество дубликатов файлов и запустите.
2. После запуска автоматиечески создастся папка **hx_settings** с файлами настроек:
- **folders.txt** — исключаемые директории.
- **file.txt** — исключаемые файлы.
- **extensions.txt** — разрешенные расширения файлов.
- **delete.txt** — режим удаления. По умолчанию null (yes для включения, null для отключения).

## Логирование
- Все операции записываются в файл process.log
